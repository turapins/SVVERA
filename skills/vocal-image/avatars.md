# Vocal Image — Avatars

Three main characters used across all Vocal Image content.
Always use reference image control — never generate a character without a reference sheet.

---

## Andy

**Role:** Primary male presenter. Professional, confident, relatable.
**Audience:** Men 25–45, business/corporate context.
**Voice (ElevenLabs):** Andy
**Visual style:** Business casual. Clean background. Eye contact with camera.
**Tone:** Direct, authoritative but warm. Gets to the point fast.
**Best for:** UGC ads, podcast host role, authority-building content.

---

## Peter

**Role:** Secondary male character. Expert/coach archetype.
**Audience:** Men 30–50, leadership/executive context.
**Voice (ElevenLabs):** Peter
**Visual style:** Smart casual or suit. Office or studio setting.
**Tone:** Mentor energy. Thoughtful, deliberate.
**Best for:** Podcast format (as guest or co-host), explainer-style content.

---

## Arya

**Role:** Female presenter. Modern, empowered professional.
**Audience:** Women 25–40, career-focused.
**Voice (ElevenLabs):** Arya
**Visual style:** Smart casual. Clean, bright environment.
**Tone:** Relatable, motivating. Speaks peer-to-peer.
**Best for:** UGC ads targeting female audience, App install campaigns.

---

## Reference Image Rules

1. Always attach a reference sheet image when generating with Higgsfield
2. One reference image per character — consistent across all scenes in a project
3. Reference scales to both photo and video generation
4. Never generate the same character twice in one project without the same reference
5. If a reference isn't available, escalate to Ivan — do not improvise
